## Portfolio - Rupesh Sahu

A professional portfolio website that highlights my profile, skills, and projects.

## Live Site
https://rupeshsahu969.github.io/

## Purpose
- Present my work and experience in one place.
- Provide fast access to my resume and contact details.
- Showcase real projects with code and live demos.

## Sections Overview
- Home: resume button, profile photo, name, and GitHub link.
- About: short summary and email contact.
- Skills: technical skills list.
- Projects: cards with source code and live demo links.
- Contact: LinkedIn, phone number, and email.

## Home
- Resume download button.
- Profile photo and name.
- GitHub profile link.

## About
- One-paragraph summary.
- Email address for direct contact.

## Skills
- Frontend: HTML, CSS, JavaScript.
- Frameworks: React.
- Tools: Git, GitHub, VS Code.

## Projects
- Each project card includes a short description.
- Source code link opens the GitHub repo.
- Live demo link opens the deployed project.

## Contact
- LinkedIn profile link.
- Phone number.
- Email address.

## Highlights
- Simple, clean layout.
- Fast navigation with clear sections.
- Mobile-friendly structure.

## Folder Notes
- `index.html` contains the main layout.
- `style.css` contains the visual design.
- `script.js` handles interactivity.

## How To Use
- Open `index.html` in a browser.
- Or visit the live site link above.

## Customization
- Update profile photo in the assets folder.
- Replace resume file with your latest PDF.
- Add new skills under the Skills section.
- Add new projects with code and demo links.

## Accessibility
- Use clear labels for buttons and links.
- Keep text contrast readable.
- Provide alt text for images.

## SEO
- Use meaningful page title.
- Add meta description.
- Keep headings structured.

## Testing Checklist
- Links open correctly.
- Resume download works.
- Project cards show correct links.
- Contact info is accurate.
